# Biologically Plausible Transformer

**A PyTorch implementation of a Linear RNN inspired by astrocyte-neuron interactions.**

A proof-of-concept Linear RNN inspired by *Kozachkov et al. (2023)*. It explores how biological mechanisms, specifically the interplay between fast neurons and slow astrocytic calcium dynamics, map to artificial neural networks, providing a biologically plausible perspective on continuous state memory.


## Overview

This project explores a biologically grounded alternative to standard attention mechanisms. By separating the computational graph into two distinct biological entities, we can model sequence processing in linear time:
1. **Neurons (Fast Non-linear Processing):** Responsible for evaluating current inputs and rapidly generating Queries, Keys, and Values.
2. **Astrocytes (Slow Linear Accumulation):** Function as the continuous state memory ($H$), buffering traces of synaptic activity over time without non-linear squashing functions.

## Architecture & Mathematics

The core of this implementation modifies the theoretical biological writing phase to include an **Exponential Decay Factor ($\gamma$)**. 

Unconstrained linear accumulation can lead to state explosion over long sequences. To stabilize the network and enforce a "recency bias" (mathematically analogous to biological calcium pumps actively removing ions from the astrocyte), we update the synaptic weight matrix $H$ at time $t$ using the following recurrence relation:

$$H_t = \gamma H_{t-1} + v_t \phi(k_t)^T$$

Where:
* $H_t$ is the astrocytic memory matrix (Continuous State).
* $\gamma \in (0, 1]$ is the exponential decay factor (e.g., $0.95$).
* $v_t$ is the Value vector generated by fast-spiking neurons.
* $k_t$ is the Key vector.
* $\phi(\cdot)$ is the biological activation function (e.g., ELU + 1) mapping to positive firing rates.

The reading phase (context extraction) is then performed via a direct matrix-vector multiplication with the Query ($q_t$):

$$y_t = H_t \cdot \phi(q_t)$$

## Key Features
* **$O(1)$ Inference Memory:** The size of the memory state $H$ remains strictly constant (`[d_model, d_model]`), regardless of the sequence length.
* **Stable Continuous State:** Thanks to the exponential decay ($\gamma$), the model can theoretically process infinite streams of tokens without exploding states.
* **Biologically Grounded:** Replaces abstract mechanisms with local synaptic learning rules and glial buffering.
